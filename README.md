# CodeAnalyzer

Code Analyzer is an analysis tool which detects syntax bugs and leaks. It supports different programming languages : Javascript, Python, C, C++, C#, Java.

Tech Stack

Front End - ReactJs, Semantic UI
Back End - Node JS, Express Js
Database - Mongo DB
